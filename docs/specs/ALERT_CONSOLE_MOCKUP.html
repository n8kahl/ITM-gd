Manage the Trade
Where the admin spends most time. The P&L bar has a prominent UPDATE button that broadcasts the current % as an informational update ("+68% here @everyone"). Trim buttons include the live P&L plus the trim instruction ("+68% here trim @everyone"). This mimics FancyITM's actual patterns: sometimes just announcing "72% here" and other times "47% here trim." The admin never types the percentage — the console knows it.
